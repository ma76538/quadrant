# Quadrant

A modern todo list application based on the Eisenhower Matrix (Four Quadrants) method, helping users prioritize tasks effectively across iOS, macOS, and Cloud platforms.

## Features

- **Four Quadrants Task Management**
  - Urgent & Important (Q1)
  - Urgent & Not Important (Q2)
  - Not Urgent & Important (Q3)
  - Not Urgent & Not Important (Q4)

- **Core Features**
  - Task creation, editing, and deletion
  - Task prioritization
  - Due dates and reminders
  - Categories and tags
  - Notes and attachments
  - Progress tracking
  - Search and filters

- **Cross-Platform Support**
  - iOS application
  - macOS application
  - Cloud synchronization
  - Real-time updates

## Technical Stack

- **Frontend**
  - iOS: SwiftUI
  - macOS: SwiftUI
  - Shared UI Components

- **Backend**
  - Firebase (Authentication, Realtime Database, Cloud Storage)
  - Cloud Functions for background tasks

- **Data Synchronization**
  - CloudKit for Apple ecosystem
  - Firebase for cross-platform sync
